Project 1 — Survey Form

Preview of the Form

This is the Survey Form project I built as part of my freeCodeCamp Responsive Web Design certification. The goal was to build a working HTML form that passes all 16 user story tests on the platform.

I learned how to use different input types like text, email, number, radio buttons, checkboxes, dropdowns, and a textarea. I also learned about the importance of labels and the HTML5 validation attributes like required, min, and max.
